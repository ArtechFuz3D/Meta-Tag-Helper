MetaTagHelper

Modern HTML meta tags reference with current best practices

Essential Core Meta Tags

The absolute minimum meta tags every modern website should have. These are non-negotiable.

<!-- Essential Core Meta Tags --> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Your Page Title (50-60 chars)</title> <meta name="description" content="Compelling description of your page content (150-160 characters max)">

Modern SEO Meta Tags

Current best practices for search engine optimization. Keywords meta tag is mostly obsolete but included for legacy support.

<!-- Modern SEO Meta Tags --> <meta name="robots" content="index, follow, max-image-preview:large, max-snippet:-1, max-video-preview:-1"> <meta name="googlebot" content="index, follow"> <meta name="author" content="Your Name or Company"> <meta name="generator" content="Your CMS or Framework"> <link rel="canonical" href="https://example.com/current-page"> <!-- Language and Locale --> <meta name="language" content="English"> <link rel="alternate" hreflang="en" href="https://example.com/en/"> <link rel="alternate" hreflang="x-default" href="https://example.com/">

Open Graph (OG) Meta Tags

Essential for social media sharing on Facebook, LinkedIn, WhatsApp, and most social platforms.

<!-- Open Graph Meta Tags --> <meta property="og:type" content="website"> <meta property="og:site_name" content="Your Site Name"> <meta property="og:title" content="Your Page Title"> <meta property="og:description" content="Your page description"> <meta property="og:url" content="https://example.com/current-page"> <meta property="og:image" content="https://example.com/og-image.jpg"> <meta property="og:image:secure_url" content="https://example.com/og-image.jpg"> <meta property="og:image:width" content="1200"> <meta property="og:image:height" content="630"> <meta property="og:image:alt" content="Descriptive text about the image"> <meta property="og:locale" content="en_US">
Tip: OG images should be 1200x630px for optimal display across all platforms.

Twitter Card Meta Tags

Optimized for Twitter sharing with improved card types and better engagement.

<!-- Twitter Card Meta Tags --> <meta name="twitter:card" content="summary_large_image"> <meta name="twitter:site" content="@yoursitehandle"> <meta name="twitter:creator" content="@yourcreatorhandle"> <meta name="twitter:title" content="Your Page Title"> <meta name="twitter:description" content="Your page description"> <meta name="twitter:image" content="https://example.com/twitter-image.jpg"> <meta name="twitter:image:alt" content="Descriptive text about the image">

Modern Security Headers

Current security best practices. These should ideally be set at the server level, but can be added to HTML as fallback.

<!-- Security Headers (Better implemented server-side) --> <meta http-equiv="X-Content-Type-Options" content="nosniff"> <meta http-equiv="X-Frame-Options" content="DENY"> <meta http-equiv="Referrer-Policy" content="strict-origin-when-cross-origin"> <meta http-equiv="Permissions-Policy" content="geolocation=(), microphone=(), camera=()"> <!-- DNS Prefetching --> <link rel="dns-prefetch" href="//fonts.googleapis.com"> <link rel="dns-prefetch" href="//www.google-analytics.com">
Note: These security headers are much more effective when implemented at the server level (Apache, Nginx, etc.) rather than in HTML meta tags.

Progressive Web App (PWA) Meta Tags
